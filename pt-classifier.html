Statistical Point Classifier

Overview

Given a dense point cloud, the classifier computes a set of thirty-nine metrics describing how points in the cloud relate to their neighbors then uses a k-means clustering algorithm to sort the points into classes corresponding to the type of object they belong to.

This project was part of my work with the forestry department at Virginia Tech, with the ultimate goal being to figure out how we can use TLS technology to assist land managers, particularly in regards to forming prescribed fire plans.

We're working on how we can use the segmented scans to estimate the volumes of various wildland fuel types such as coarse woody debris, loose leaf litter, and duff.
